# Шаблон для работ в ВлГУ (Word, ГОСТ 7.32-2017)

**Основано на https://github.com/konstantin-morenko/report-732-2017**

## Порядок использования

1. Импортируйте файл `report.exportedUI` (Файл -> Параметры -> Настроить ленту -> Импорт/экспорт -> Импортировать файл)
2. Поместите файл `Отчёт.dotm` в  `C:\Пользователи\<Имя пользователя>\Документы\Настраиваемые шаблоны Office`
3. Создайте новый документ из шаблона и готово!
